# Sound Assets

Place your sound files in this directory with the following names:

- `intro.mp3` - Intro/menu background music
- `jump.mp3` - Jump sound effect
- `slash.mp3` - Katana slash/attack sound
- `hit.mp3` - Impact/hit sound effect
- `land.mp3` - Landing sound effect

## Recommended Sound Specifications
- Format: MP3 or OGG
- Sample Rate: 44.1kHz
- Bit Rate: 128-192 kbps
- Duration: 0.5-2 seconds (except intro)

## Free Sound Resources
- Freesound.org
- OpenGameArt.org
- Zapsplat.com
- Mixkit.co

## Volume Levels (configured in sound.js)
- Intro: 30%
- Jump: 40%
- Slash: 50%
- Hit: 60%
- Land: 40%
